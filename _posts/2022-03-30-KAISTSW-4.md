### 1. True, False : 이분법적 분기 (if, else)

```py
x = int(input())

if x%2 == 0 :
    print("짝수")

else :
    print("홀수")
```

<br>

- 문법 정리

<br>

```
if 조건식 : ==> 반드시 True or False로 판별되어야 함
    (조건이 참일 경우 실행할 코드)

else :
    (조건이 거짓일 경우 실행할 코드)
```

### 2. 다중 분기 : 2개 이상의 경우의 수로 분기되는 경우

<br>

```py
no = int(input())

# W : 슛 A : 달리기 S : 태클 D : 정지

if key == 'W':
    print("슛")

else :
    if key == "A":
        print("달리기")
    else :
        if key == "S": # W or A 가 아닌 나머지 모든 경우
            print("태클")
        else : 
            if key == "D":
                print("정지")

                else :
                    print("아무 동작 X)
```

&uarr; 아주 안 좋은 코드 !!

<br>

```py
# elif 활용 (else + if) 
if key == "W":
    print("슛")
elif key == "A":
    print("달리기")
elif key == "S":
    print("태클")
elif key == "D":
    print("정지")
else : 
    print("아무 동작 X")
```
&uarr; **가독성**이 좋은 코드 !!

### While : 프로그래밍 언어의 최초의 반복문

<br>

```py
while 조건식:
    실행코드

== 1부터 10까지 화면에 출력해라. ==

# 초기값 : 1, 조건 : <=10, 증감 : 1씩 증가

i = 1 #  초기값

while i <= 10 : # 조건
    print(i)
    i += 1 # 증감
```
```py
for i in range(1, 11, 1) : # 초기값, 조건, 증감
    print(i)
```

<br>

```
Ex_ 구구단 7단을 화면에 출력.
```
```py

    print(i)
```
```py
for i in range(1, 10,1):
    print(7 * i)
```
